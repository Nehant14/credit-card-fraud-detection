# Credit Card Fraud Detection ML

Machine Learning system to detect fraudulent credit card transactions.

## Dataset

Download dataset:

https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

Place it inside:

data/creditcard.csv

## Installation

pip install -r requirements.txt

## Run Full Project

python run_project.py


## Algorithms

Isolation Forest  
Gradient Boosting

## Skills Demonstrated

Fraud Detection  
Imbalanced Dataset Handling  
ML Pipelines  
Data Visualization
