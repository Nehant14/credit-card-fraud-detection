## Dataset

This project is based on the **Credit Card Fraud Detection dataset** available on Kaggle:

https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

The original dataset contains **284,807 transactions with 31 features** and is about **150 MB in size**. Since this is quite large for a GitHub repository, only a **small sample dataset** is included in this project to demonstrate the pipeline and allow quick testing.

For full training and accurate results, please download the original dataset from Kaggle and place the file:

creditcard.csv

inside the following directory:

data/raw/creditcard.csv
